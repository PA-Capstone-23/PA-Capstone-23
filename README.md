a. Kunjungan
 - mendeklarasi package di mana kelas Kunjungan berada.
 ![image](https://github.com/PA-Capstone-23/PA-Capstone-23/assets/122289679/05671aec-6baa-498c-8025-ccf9b729c4eb)
